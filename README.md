# CMOD-S7-MIPS-SINGLE

Digilent Cmod S7-25 보드를 사용하여 MIPS Pipeline Processor를 구현합니다.


## 개발 환경

- Board: Digilent Cmod S7-25
- FPGA: Xilinx Spartan-7
- Tool: Xilinx Vivado 2019.2
- HDL: Verilog


## 프로젝트 목표

logisim으로 미리 작성된 pipeline MIPS를 기반으로 모듈을 RTL 설계하고, 기능을 검증하고, 타이밍을 검증한다.
이후 이전의 Single cycle MIPS와 타이밍 기반 비교한다.
